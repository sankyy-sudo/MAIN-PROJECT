# COTECAE CRM & B2B E-Commerce Platform

Production-oriented MERN stack platform built with React, TypeScript, Express, MongoDB, JWT authentication, RBAC, and CRM workflows.

## Structure

```txt
crm-platform/
├── frontend/
├── backend/
├── shared/
├── docs/
├── package.json
└── README.md
```

## Current Completion

Phase 1 authentication, dashboard, user management, protected routing, Redux setup, and dashboard widgets are wired.

Phase 2 CRM lead and customer management APIs and frontend pages are wired, including lead filters/search/pagination, lead activity timeline, and customer notes/documents timeline endpoints.

## Commands

```bash
cd backend
npm run build
npm run dev
```

```bash
cd frontend
npm run build
npm run dev
```
